## 💡 Was ich gelernt habe

### ServerScriptService vs. Script im Workspace
- Scripts direkt unter einem Part im Workspace sind für den Client sichtbar → Sicherheitsrisiko
- Bei `Destroy()` des Parts stirbt das Script mit
- `ServerScriptService` = sicherer, sauberer, professioneller

### Zwei For-Loop Typen

**Numerischer For-Loop** (wie `for` in C#):
```lua
for i = 0, 5, 1 do   -- start, end, schritt
    -- zählt: 0, 1, 2, 3, 4, 5
end
```

**Iterator For-Loop** (wie `foreach` in C#):
```lua
for _, part in partsArray do
    -- '_' = Index wird ignoriert (Konvention)
end
```

### Tables als Arrays
```lua
local blocks = {
    workspace.Block1,
    workspace.Block2,
}

#blocks        -- Länge der Table (wie .Length in C#)
blocks[1]      -- Zugriff per Index (WICHTIG: startet bei 1, nicht 0!)
```

### GetChildren()
```lua
local folder = workspace.MeinFolder
local parts = folder:GetChildren()  -- alle Children als Table
-- Vorteil: neuen Block in Folder ziehen = kein Code ändern nötig
```

### Negative Schrittweite
```lua
for i = 5, 0, -1 do  -- runterzählen
    -- ohne '-' würde der Loop einfach übersprungen
end
```

### Konfigurationsvariablen (Best Practice)
```lua
local VANISH_DELAY = 1.5   -- GROSSBUCHSTABEN = Konstante (Konvention)
local timePerChange = 0.3
-- → Magic Numbers vermeiden, einmal ändern gilt überall
```

## 🔗 Vergleich zu C#

| Luau | C# |
|---|---|
| `for i = 0, 5, 1 do` | `for (int i = 0; i <= 5; i++)` |
| `for _, x in array do` | `foreach (var x in array)` |
| `#array` | `array.Length` |
| `array[1]` | `array[0]` (Index startet bei 1 statt 0!) |
| `workspace` | globaler Shortcut für `game.Workspace` |

## 🏗️ Was ich gebaut habe

**Danger Zone Obby** — Disappearing Path mit Warnlicht:
- 🟢 Blöcke starten grün und sichtbar
- 🔴 Jeder Block wird einzeln rot, dann unsichtbar + `CanCollide = false`
- 💡 SpotLight pulsiert (0→5→0) jedes Mal wenn ein Block verschwindet
- 🔄 Nach allen Blöcken: Reset, alles startet von vorne

```lua
local partsFolder = workspace.Folder
local partsArray = partsFolder:GetChildren()
local brightness = 1
local timePerChange = 1
local light = workspace.WarnLight

while true do
    for _, part in partsArray do
        part.Color = Color3.fromRGB(55, 255, 59)
        part.Transparency = 0
        task.wait(2)
        part.Color = Color3.fromRGB(255, 54, 37)
        task.wait(2)
        part.Transparency = 1
        for currentBrightness = 0, 5, brightness do
            light.Brightness = currentBrightness
            task.wait(timePerChange)
        end
        for currentBrightness = 4, 0, -brightness do
            light.Brightness = currentBrightness
            task.wait(timePerChange)
        end
        task.wait(2)
    end

    for _, part in partsArray do
        part.Color = Color3.fromRGB(55, 255, 59)
        part.Transparency = 0
    end
end
```
