## 🔄 Luau vs. C# — Schnellreferenz

| Konzept | C# | Luau |
|---|---|---|
| Variable | `int x = 5;` | `local x = 5` |
| Funktion | `void Greet() {}` | `local function greet() end` |
| Klasse | `class Player {}` | `local Player = {}` (Table) |
| Null | `null` | `nil` |
| String | `"Hello"` | `"Hello"` |
| Boolean | `true / false` | `true / false` |
| Array | `int[] arr` | `local arr = {}` (Table) |
| For-Schleife | `for (int i=0; i<10; i++)` | `for i = 0, 9 do` |
| Kommentar | `// Kommentar` | `-- Kommentar` |
